pd sirf ek short name (alias) hai.

Step 2: Series kya hoti hai?

Series ek single column ki tarah hoti hai.

Example:

import pandas as pd

numbers = pd.Series([10, 20, 30, 40])

print(numbers)

Output:

0    10
1    20
2    30
3    40
dtype: int64

Yahan:

Left side = Index
Right side = Value
Step 3: Apni Index Banana
import pandas as pd

marks = pd.Series(
    [80, 90, 95],
    index=["Ram", "Shyam", "Mohan"]
)

print(marks)

Output:

Ram      80
Shyam    90
Mohan    95
dtype: int64
Step 4: Kisi ek Value ko Access Karna
print(marks["Ram"])

Output:

80
