📂 Topic: Reading CSV Files (Real-World Data)

Aaj se hum real data ke saath kaam karna shuru karenge. Companies me 90% data CSV, Excel ya Database se aata hai.

📌 CSV Kya Hota Hai?

CSV = Comma Separated Values

Example (students.csv):

Name,Age,City,Marks
Satyam,18,Khajuraho,90
Shivam,20,Indore,85
Lakshya,18,Bhopal,88

Excel me bhi ise open kar sakte ho.

Step 1 – CSV Read Karna

Sabse pehle ek file banao.

students.csv

Name,Age,City,Marks
Satyam,18,Khajuraho,90
Shivam,20,Indore,85
Lakshya,18,Bhopal,88
Rahul,19,Delhi,80

Ab Python me:

import pandas as pd

df = pd.read_csv("students.csv")

print(df)
📌 Samjho
pd.read_csv("students.csv")

Iska matlab:

➡️ "students.csv file ko read karo aur DataFrame bana do."

Step 2 – First 3 Rows
print(df.head(3))

Output

      Name  Age        City  Marks
0   Satyam   18  Khajuraho     90
1   Shivam   20     Indore     85
2  Lakshya   18     Bhopal     88
Step 3 – Last 2 Rows
print(df.tail(2))
Step 4 – Column Access
print(df["Marks"])
Step 5 – Multiple Columns
print(df[["Name", "Marks"]])
Step 6 – Shape
print(df.shape)

Output

(4,4)
Step 7 – Columns
print(df.columns)
Step 8 – Data Types
print(df.dtypes)

Output

Name     object
Age       int64
City     object
Marks     int64

👉 object ka matlab generally text (string) hota hai.

Step 9 – Information
print(df.info())

Ye batata hai:

Rows
Columns
Missing Values
Data Type
Memory
Step 10 – Basic Statistics
print(df.describe())

Ye sirf numeric columns ka summary deta hai:

Count
Mean
Min
Max
Standard Deviation
