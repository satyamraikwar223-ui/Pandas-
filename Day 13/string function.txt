Welcome to Day 13! 🚀

Aaj hum Pandas ke String Functions seekhenge.

Ye topic Data Cleaning, NLP (Natural Language Processing) aur AI me bahut important hai.

🤔 Real-Life Example

Maan lo company ke database me names aise stored hain:

Name
" satyam "
"RAHUL"
"priya"
"NeHa"

AI ya Data Analysis se pehle hume inhe clean karna padega.

Step 1 – DataFrame
import pandas as pd

df = pd.DataFrame({
    "Name": [" satyam ", "RAHUL", "priya", "NeHa"],
    "City": ["Delhi", "indore", "BHOPAL", "gwalior"]
})

print(df)
1️⃣ .str.upper()

Sabko uppercase me convert karo.

print(df["Name"].str.upper())

Output:

0     SATYAM
1      RAHUL
2      PRIYA
3       NEHA
2️⃣ .str.lower()

Sabko lowercase me.

print(df["Name"].str.lower())

Output:

0    satyam
1     rahul
2     priya
3      neha
3️⃣ .str.title()

Har word ka first letter capital.

print(df["Name"].str.title())

Output:

Satyam
Rahul
Priya
Neha
4️⃣ .str.strip()

Extra spaces hata do.

print(df["Name"].str.strip())

Output:

satyam
RAHUL
priya
NeHa
5️⃣ .str.replace()

Text replace karo.

print(df["City"].str.replace("Delhi", "Mumbai"))

Output:

Mumbai
indore
BHOPAL
gwalior
6️⃣ .str.contains()

Check karo kisi string me word hai ya nahi.

print(df["City"].str.contains("ind"))

Output:

0    False
1     True
2    False
3    False

Sirf matching rows:

print(df[df["City"].str.contains("ind", case=False)])

case=False → uppercase/lowercase ignore karega.

7️⃣ .str.startswith()
print(df["Name"].str.startswith("R"))

Output:

False
True
False
False
8️⃣ .str.endswith()
print(df["Name"].str.endswith("a"))
9️⃣ .str.len()

String ki length.

print(df["Name"].str.len())
🔟 Multiple Operations
df["Name"] = df["Name"].str.strip().str.title()

print(df)

Ye industry me bahut common pattern hai.

    "Name": [" aman ", "ROHIT", "neha", "PRIYA"],
    "Department": ["hr", "IT", "finance", "HR"]

Ab ye sab likho:

print(employee["Name"].str.upper())

print(employee["Name"].str.lower())

print(employee["Name"].str.title())

print(employee["Name"].str.strip())

print(employee["Department"].str.upper())

print(employee["Department"].str.lower())

print(employee["Department"].str.replace("HR", "Human Resource"))

print(employee["Name"].str.len())

print(employee["Name"].str.startswith("A"))
