🎉 Pandas Course – Day 6/30
📍 Topic: Sorting Data (sort_values() & sort_index())

Aaj hum seekhenge Sorting. Real-world me data ko ascending (A→Z, small→large) ya descending (Z→A, large→small) order me arrange karna bahut common task hai.

Examples:

Highest marks wale students
Lowest salary wale employees
Alphabetically sorted names
Step 1 – DataFrame
import pandas as pd

student = {
    "Name": ["Satyam", "Shivam", "Lakshya", "Rahul", "Priya"],
    "Age": [18, 20, 19, 21, 18],
    "City": ["Khajuraho", "Indore", "Bhopal", "Delhi", "Indore"],
    "Marks": [90, 85, 88, 92, 95]
}

df = pd.DataFrame(student)

print(df)
1️⃣ Marks ke hisaab se Sort (Ascending)
print(df.sort_values("Marks"))

Output:

Shivam   85
Lakshya  88
Satyam   90
Rahul    92
Priya    95

Default sorting ascending hoti hai.

2️⃣ Marks ke hisaab se Descending
print(df.sort_values("Marks", ascending=False))

Output:

Priya    95
Rahul    92
Satyam   90
Lakshya  88
Shivam   85
3️⃣ Name Alphabetically Sort
print(df.sort_values("Name"))

Output:

Lakshya
Priya
Rahul
Satyam
Shivam
4️⃣ Age ke hisaab se Sort
print(df.sort_values("Age"))
5️⃣ Multiple Columns se Sort

Pehle City, phir Marks.

print(df.sort_values(["City", "Marks"]))

Agar ek hi city ke multiple students honge, to unhe Marks ke hisaab se sort karega.

6️⃣ Mixed Sorting

City → A to Z

Marks → High to Low

print(df.sort_values(["City", "Marks"], ascending=[True, False]))
7️⃣ Original DataFrame ko Permanently Sort Karna
df.sort_values("Marks", inplace=True)

print(df)

⚠ inplace=True ka matlab original DataFrame change ho jayega.

8️⃣ Index ke hisaab se Sort
print(df.sort_index())

Descending:

print(df.sort_index(ascending=False))
9️⃣ Top 3 Students
print(df.sort_values("Marks", ascending=False).head(3))

Output:

Priya
Rahul
Satyam
🔟 Lowest 2 Marks
print(df.sort_values("Marks").head(2))
